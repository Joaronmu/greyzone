# Mixamo 绑骨 + 动画 上传指引

## 你要上传的文件（已减面到 <7万面，A-pose，身高1.8m，居中）
- `enemy_for_mixamo.obj`   → 第一关 y目标（敌人）
- `miniboss_for_mixamo.obj` → 小 boss
- `mainboss_for_mixamo.obj` → 主 boss（z目标）

> 机枪和垃圾桶不需要动画（机枪只旋转、桶是静态道具），不用上传。

## 步骤（每个角色都做一遍）

### 1. 上传
1. 打开 https://www.mixamo.com （需 Adobe 账号，免费）
2. 右上角 **Upload Character** → 选 `xxx_for_mixamo.obj`
3. 等它处理（几秒~十几秒）。如果报 "unable to match skeleton"，多半是站姿识别问题——见下方【常见问题】。

### 2. 自动绑骨（Auto-Rig）
1. 上传后会进入 **Character 预览**，自动进入绑骨流程
2. 它会让你确认/拖动几个关节标记点：下巴、手腕、手肘、膝盖、胯、脊椎等
3. 你的角色是 **A-pose（手臂下垂）**，所以手腕标记点在身体两侧下方，别拖成 T-pose 的水平位置
4. 点 **Next** 直到 **Rig** 完成生成骨骼（65根 mixamorig 骨头）
5. 预览里能看到角色动了（默认待机）就说明绑骨成功

### 3. 加动画
1. 顶部切到 **Animations** 标签
2. 搜索你想要的动画，例如：
   - `Rifle Walk` / `Rifle Walk To Stop`（你下载过的那个）
   - `Idle` / `Rifle Idle`
   - `Run` / `Rifle Run`
   - `Death` / `Dying`
3. 点动画 → 预览里你的角色会做这个动作
4. 右侧 **Download** 设置：
   - **Format**: `FBX for Unity`（或 `FBX`）
   - **Skin**: **`With Skin`**（必须！这样下载的 FBX 才带网格+骨骼+权重+动画）
   - **Frames per second**: 30
   - **Keyframe Reduction**: `None`（保留全部关键帧，动作更准）
5. **Download** → 保存为 `enemy_walk.fbx`、`enemy_idle.fbx` …（一个动画一个文件）

### 4. 重复
- 每个角色（敌人/小boss/主boss）都做一遍"上传→绑骨→下载动画"
- 同一个角色绑骨一次后，**之后换动画不用重新绑骨**——直接在 Animations 里挑别的动画下载即可（都带同一套骨骼）
- 建议每个角色至少下载：**行走(Rifle Walk)**、**待机(Rifle Idle)**、**死亡(Death)** 三种

## 常见问题

**Q: 上传报错 "unable to match your existing skeleton" / "Sorry, unable to..."**
- OBJ 没骨架不会出这个错（这是给已绑骨模型用的）。OBJ 上传一般不会报这个。
- 如果上传失败，检查文件没损坏——这三个文件已验证 v/vt/vn/f 完整。

**Q: 自动绑骨后角色扭曲/姿势不对**
- 多半是标记点放错。A-pose 时：手腕在身体两侧大腿附近，手肘在手腕上方，膝盖在大腿下方。重新拖标记点。

**Q: 想用 T-pose 效果更好？**
- 可以，但你的模型是 A-pose。Mixamo 对 A-pose 支持没问题，绑骨后在动画里会自动纠正到动画姿态，不用手动改 T-pose。

**Q: 面数会不会太高被拒？**
- 三个文件都 <6.4万面，Mixamo 上限约 7万面，没问题。
